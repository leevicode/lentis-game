# lentis-game
A port of a work in progress volleyball game from my home page

## Goals
- The primary goal is to port the game to C or another similar rigid compiled language.
- The main goal is to produce a fully playable experience with either local or online multiplayer.
- An additional goal is to implement 3D-graphics to the game.

## Prerequisites
Raylib is needed. Place headers in include/ directory.
Libraries are placed in lib/ directory.
